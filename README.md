Hur ni arbetar ihop (Arbetsflödet)
När ni skriver kod använder ni tre huvudkommandon i CLion:

Commit: Sparar dina lokala ändringar på din egen dator med ett meddelande om vad du gjort.

Push: Skickar dina sparade ändringar från din dator upp till GitHub så din vän kan se dem.

Pull: Hämtar de senaste ändringarna från GitHub som din vän har skickat upp.

Tips: Gör alltid en Pull innan du börjar skriva kod för att se till att du har den senaste versionen. (JetBrains Documentation)
